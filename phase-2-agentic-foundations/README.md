# Phase 2 — From Generative to Agentic (Days 11–22)

The leap: turn a text generator into something that **observes → decides → acts → repeats**.
You build a tool-using agent **from scratch** (no framework) so you understand every moving part.

**Prereqs:** Days 11–13 need no API key. From Day 14, configure a provider
([../SETUP.md](../SETUP.md)). Uses the [`shared/`](../shared) toolkit.
